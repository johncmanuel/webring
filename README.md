# FartLabs Webring

A webring for FartLabs members and friends. Hand-crafted sites welcome.

## Join the Webring

1. Fork this repo
2. Edit `index.html` and add yourself to the `<ol id="icons">` list:
   ```html
   <li data-lang="en" id="your-handle">
       <a href="https://your-site.com">your-name</a>
   </li>
   ```
3. Submit a pull request

## Rules

- Hand-crafted wikis, portfolios, and personal sites only
- No JS required (but fine if your site uses it)
- Be excellent to each other

## Member Sites

Visit [webring.fartlabs.com](https://webring.fartlabs.com) to browse the ring.
